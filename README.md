Diplom_1

Автоматизация unit-тестов для приложения Stellar Burgers.

Проект посвящен тестированию логики создания бургеров с использованием unit-тестов, моков и параметризации.

## Используемые технологии

* Java 11
* Maven
* JUnit 4
* Mockito
* JaCoCo

## Архитектура проекта

Тестирование построено с использованием:

* unit-тестов;
* моков и стабов Mockito;
* параметризованных тестов;
* проверки покрытия кода через JaCoCo.

## Реализованные проверки

Покрыты тестами:

* класс Burger;
* работа ингредиентов;
* расчет стоимости;
* отображение состава бургера.

## Особенности проекта

* Использована параметризация тестов;
* Использованы моки для изоляции зависимостей;
* Достигнуто 100% покрытие кода.

## Запуск тестов

mvn clean test

## Генерация отчета о покрытии

mvn jacoco:report
